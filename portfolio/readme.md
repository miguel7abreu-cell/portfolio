# 📊 Miguel Abreu | Portfolio: Lead Business Analyst & BI Specialist

Este repositorio contiene el código fuente de mi portfolio profesional. El sitio está diseñado bajo una filosofía de **"Conversión Directa"**, optimizado para que Recruiters y Stakeholders validen mi propuesta de valor en menos de 90 segundos.

---

## 🎯 Objetivo del Sitio
Demostrar mi capacidad híbrida para liderar el descubrimiento de negocio (BA) y la ejecución técnica de datos (BI), manteniendo una estética premium y una navegación fluida.

## 🛠️ Stack Tecnológico
* **Frontend:** HTML5 semántico, CSS3 (Flexbox/Grid), JavaScript Vanilla (ES6+).
* **Performance:** Carga diferida (Lazy-load), sin librerías pesadas (No jQuery, No React).
* **UX/UI:** Sistema de *Drawer* para Case Studies para mantener la retención del usuario en una sola página.
* **Accesibilidad:** Cumplimiento de estándares WCAG AA.
